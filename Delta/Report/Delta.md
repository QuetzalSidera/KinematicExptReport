# 实验四：Delta 并联机器人系统控制

## 基本信息

- 开课学期：2026 春
- 课程名称：机器人机构学
- 实验名称：Delta 并联机器人系统控制

## 一、实验目的

1. 掌握 Delta 并联机器人系统的控制方法与应用。
2. 掌握不同控制模式下的控制技术。
3. 掌握 Python 程序的编写方式，能够编制控制程序。
4. 通过实验台掌握用 PyCharm 编程控制 Delta 并联机器人系统，达到控制运动的目的。

## 二、实验内容

通过机器人关节连接连杆带动末端平面，从而控制末端笔尖运动。由 PyCharm 软件控制 Delta 并联机器人运行和停止，程序使用 Python 语言，通过 USB 转 CAN 通讯模块将控制程序指令发送给机器人，控制末端运动到不同的位置，并可对照坐标板来测量位置；或首先规划机器人的运动轨迹，得到要运动的所有坐标点位，控制末端执行规划的轨迹。

## 三、实验设备和工具

1. 计算机
2. Delta 并联机器人
3. DP3010A 学生电源
4. 急停开关
5. USB-CAN 通讯模块
6. PyCharm 编程软件
7. Python 3.10 编程语言环境
